## Project Overview

The Team Member Management App is a full-stack web application built to manage and organize member information efficiently. The application allows users to add member details, store them securely in a database, and view saved records through an interactive user interface.

It demonstrates complete integration between frontend, backend, and database technologies.

## Features

- Add new member details  
- View all saved members  
- Display detailed member information  
- User-friendly React interface  
- Backend API with Node.js and Express.js  
- MongoDB database integration for persistent storage  

## Technologies Used

### Frontend
- React.js  
- React Router  
- CSS 

### Backend
- Node.js  
- Express.js  

### Database
- MongoDB  
- Mongoose
